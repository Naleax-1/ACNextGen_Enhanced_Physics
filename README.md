# ACNextGen_Enhanced_Physics
This module extends and adds to Assetto Corsa's physics, and also allows anyone to edit the physics. 
 Once downloaded, drag and drop it into the Content Manager; it will be downloaded to assettocorsa/apps/lua. 
 This appscript was developed in collaboration with GPT 5.5. Therefore, we do not intend to distribute it for a fee in the future.
